Welcome to my page. This page was created as part of a web development class at Oregon State University during fall of 2019. This website uses only HTML, CSS, and vanilla javascript for all of its features.

The MVC design for the central feature of this website was developed using freeCodeCamp.org's video on creating a platformer JS game for inspiration and as a reference.

The characters that fall as part of the game are not actually randomly generated but are taken from the Lorem Ipsum text in the box follow this sentence. The game will iterate through the text and drop the next letter and start again at the beginning if the user is playing long enough to exhaust the content.
